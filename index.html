Functions

Lets began our trip together with Functions.

They are one the fundamental building blocks of JavaScript.

A function is a set of statements or conditionals that performs a certain task.

To use a function, you must define it somewhere in the scope from which you wish to call it.

function name(parameters){ console.log("This is a declaration") };

This is an example of what a function may look like. The function begins with naming of the function. Then it is follwed by a list of parameters, enclosed in parentheses and separated by commas. Lasty, there are JavaScript statements that define the function and are enclosed within curly brackets, { }.

Objects

Next up, we have our Objects.

Like in real everyday life, ojects have different properties like weight and color as well as differet methods such as starting and stopping.

Objects are like variables. The only differece is onjects contain many values rather than one.

You create an object like you would a variable by calling a name and asigning it values.

let garden = { vegetable: 'zucchini', flower: 'sun flower', fruit: 'grape', water: true, sun: true, size: 10, };

This is an example of what an object may look like. Creating an object begins with naming a container and then asigning properties and their values. Objects are extremly useful for storing a lot of data into a single container.

Arrays

Lastly, we have found ourselves at the Arrays

Arrays are global objects which are just high-level list-like objects

let students = ["Tony", "Marshall", "Rhys", "Ray", 23, true];

This is an example of what an array may look like. Since it's an object, you first began with naming a containter and then asign it just values rather properties and their values. Instead of using parentheses () you use brakets []. In JavaScript, seeing brakets is how you know an oject is an array.
